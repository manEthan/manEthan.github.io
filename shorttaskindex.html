PhoneGap



  • Developed using HTML5, CSS, JavaScript
  • Apps are downloadable from Google Play or Apple App Store
  • Able to process HTML elements
  • Useful for apps needing features using smartphone cameras, sensors, GPS, and other hardware
  • Better access to Native APIs
  • Has UI libraries that improve the user interface
  • Doesn't have the look and feel of native apps

Flutter



  • Developed using Dart programming language, which is simple and effective
  • Used to create apps that give native UI experience for both Android and iOS platforms
  • Renders every view component, allowing for applications with native-like performance characteristics
  • Draw from components of native Android and iOS libraries
  • Apps look like native iOS and Android apps, using platform-specific themes



React Native



  • Developed using React JS framework
  • No need to learn languages like Java or Objective-C
  • Have a native look and feel
  • Smooth, fast animations with faster response times
  • Some plugins require knowledge of native language(Java, Swift, Objective-C) and IDE(Xcode, Android Studio)
  • Used more than any other cross-platform mobile framework
